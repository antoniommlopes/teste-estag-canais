Execução via prompt

Insira os arquivo ".java" (pasta src) e o arquivo "entrada.txt" dentro da pasta que desejar consultar através do prompt, execute o comando de compilação para todos os arquivos java
javac *.java

Após a compilação basta executar o programa "MyProgram", através do comando
java MyProgram



Para execução através de IDE, basta abrir o projeto na ide desejada (foi utilizado o intelliJ para desenvolvimento) e executar. É necessário que o arquivo "entrada.txt" 
esteja na raiz do diretório.
